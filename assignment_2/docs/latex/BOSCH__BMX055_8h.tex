Executes the wake-\/up sequence for the BMX055 magnetometer. 


\begin{DoxyItemize}
\item Forces the magnetometer interface to switch from I2C to SPI by toggling the MAG\+\_\+\+CS pin.
\item Transitions the magnetometer from Suspend Mode to Sleep Mode by writing 0x01 to register 0x4B.
\item Transitions the magnetometer from Sleep Mode to Active Mode by writing 0x00 to register 0x4C.
\item Utilizes TIMER1 delays between transmissions to allow for power mode switching times.

void BMX055\+\_\+setup (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Initializes the BMX055 sensor and configures the SPI bus parameters. 


\begin{DoxyItemize}
\item Sets Chip Select (CS) lines for the Accelerometer, Gyroscope, and Magnetometer to high (inactive).
\item Temporarily disables the SPI module to configure protocol settings matching the BMX055 requirements.
\item Configures the SPI clock polarity to idle LOW (CKP = 0) and sets data changes to occur on transition from active to idle (CKE = 1).
\item Sets the accelerometer\textquotesingle{}s initial bandwidth to 1000 Hz by writing value 15 to register 0x10.
