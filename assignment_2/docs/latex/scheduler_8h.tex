\doxysubsection{Detailed Description}
Cooperative non-\/preemptive task scheduler header for the robot buggy. 

\begin{DoxyAuthor}{Author}
Simone Canella, Samuele Isaia, Riccardo Parotelli 
\end{DoxyAuthor}
\begin{DoxyVersion}{Version}
1.\+0.\+0
\end{DoxyVersion}
This module declares a lightweight cyclic scheduler operating on a fixed time base. It maintains a tracking matrix of registered system tasks and updates execution flags at every interval tick. Tasks execute completely to completion without nested thread interruption.

NOTE:\+
\begin{DoxyItemize}
\item The system tick interval is centralized here.
\item All registered tasks must execute quickly to prevent timing overruns on subsequent tasks. 
\end{DoxyItemize}
