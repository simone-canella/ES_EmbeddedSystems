\doxysubsection{Detailed Description}
Hardware timer configuration and delay management routines. 


\begin{DoxyItemize}
\item This file implements the setup and control functions for hardware timers TIMER1 through TIMER5. It calculates prescaler and period register values dynamically based on a 72 MHz system instruction frequency (Fcy) to provide accurate timing and blocking delays. 
\end{DoxyItemize}
