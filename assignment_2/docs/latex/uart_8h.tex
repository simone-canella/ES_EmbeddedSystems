Hardware interrupt service routine for UART1 reception. 

Triggered asynchronously when incoming bytes arrive. It handles hardware overrun errors (OERR) by clearing the bit (which discards corrupted data), and safely copies valid incoming bytes into the RX software circular buffer. If the circular buffer is full, incoming data is silently dropped to prevent memory corruption. \Hypertarget{uart_8h_a1462a0d94f37f67118aed180ad626d9b}\index{uart.h@{uart.h}!\_U1TXInterrupt@{\_U1TXInterrupt}}
