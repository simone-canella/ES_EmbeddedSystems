void spi1\+\_\+mikrobus1\+\_\+setup (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Configures the SPI1 peripheral pins, control registers, and default chip selects. 


\begin{DoxyItemize}
\item Maps the MISO line to input pin RA1 (RPI17).
\item Maps the MOSI line to output pin RF13 (RP109).
\item Maps the SCK (Serial Clock) line to output pin RF12 (RP108).
\item Initializes three independent Chip Select (CS) pins (RB3, RB4, RD6) as digital outputs idling HIGH (inactive).
\item Configures the SPI core for Master Mode, 8-\/bit data width, and sets prescalers to control the SCK frequency.
\item Enables the SPI1 module.
