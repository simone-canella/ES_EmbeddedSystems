void scheduler ( )

Core scheduler routine that evaluates and executes ready tasks.

Main task evaluation and execution loop handler.

Core scheduler routine that evaluates and executes ready tasks.

Returns
void

This routine must be called inside the main infinite processing loop. It iterates through the matrix of registered tasks, decrements their remaining delay counts, and calls the function pointers of active tasks whose intervals have expired.
